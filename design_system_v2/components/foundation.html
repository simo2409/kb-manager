Foundation

Colors, typography, spacing and core tokens.

Typography

The type scale uses Biennale font with weights from 100 to 600.

H1 — Heading (38px, weight 600)

H2 — Heading (28px, weight 400)

H3 — Heading (23px, weight 400)

H4 — Heading (23.75px, weight 400)

H5 — Label uppercase (19px, weight 500)

H6 — Small label uppercase (14px, weight 500)

Lead text — italic, 23.75px, weight 400

Body text — 19px, weight 400, line-height 1.6. Used for paragraphs and general content.

Small text — 14px, weight 400. Used for captions, meta info.

Caption — 12px, weight 400, muted color.

Blockquote — italic, left border accent, used for quotes.

Inline code: const x = 42;
