Cards

Containers for content with glassmorphism, flat and interactive variants. Hover is opt-in via ds-card--hover.

Static vs Hover

By default cards are static. Add ds-card--hover to enable the lift-and-glow effect on mouse over.

Static Card

No hover effect. Use for dashboards, read-only lists or dense grids where motion would be distracting.

Hover Card

Lifts and glows on hover. Ideal for galleries, product grids and any clickable surface that invites exploration.

Basic Card

Default glassmorphism card with optional hover lift effect.

Flat Card

Solid background without glassmorphism blur. Combine with ds-card--hover when you want motion.

Terms of Service

This flat card variant uses a solid elevated background without backdrop blur. It's ideal for long-form content, terms pages, or sections where glassmorphism would create visual noise.

Selected

Use ds-card--selected to highlight one or more cards in a list or grid. The state combines a tinted background, a thicker accent border and a subtle lift so it is perceivable even without colour vision. Combine with aria-selected="true" when the card acts as a selectable gridcell.
