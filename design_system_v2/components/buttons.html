Buttons

Interactive button components with multiple variants, sizes and states.

Variants

Primary, secondary, outline, ghost and danger styles.

Sizes

Small, default and large button sizes.

States

Disabled and full-width states.

With Icons

Buttons can include icons and icon-only variants.

Split Button

A button with a primary action and a dropdown toggle. Click the label for the main action, or the arrow to reveal alternate options.

Interaction

All buttons have hover, active and focus-visible states.
