Design System v2

A dark-first design system derived from the Simone Dall'Angelo brand identity. Built for clarity, elegance and mobile-first experiences.

Principles

  • Contrast first Every text must be readable on dark backgrounds
  • Glassmorphism Cards use semi-transparent layers with subtle borders
  • Minimal motion Transitions are soft and purposeful (150–350ms)
  • Mobile native Touch targets ≥ 44px, grids collapse to 1 column
  • Semantic color Tokens map to meaning: info, success, warning, danger

Architecture

The system is organised in 7 component categories, each with a dedicated reference page. Every component below carries a unique ID (e.g. DS-BTN-001) for traceability across design, development and QA.

Typography Scale

The type scale is anchored on Biennale, a geometric sans-serif. Weights range from 100 (ultra-thin display) to 600 (bold headings). Line-height is generous (1.2–1.5) to maintain airiness on dark backgrounds.
